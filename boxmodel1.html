BOX MODEL

In CSS, the term "box model" is used when talking about design and layout.
The CSS box model refers to how HTML elements are modeled in browser engines and how the dimensions of those HTML elements are derived from CSS properties.
It is a fundamental concept for the composition of HTML webpagesThe CSS box model is essentially a box that wraps around every HTML element.
It consists of:

  • Margins
  • Borders
  • Padding
  • Actual Content

  • width: auto;
  • margin: 100px.
  • height: auto;
  • border: 10px ;
